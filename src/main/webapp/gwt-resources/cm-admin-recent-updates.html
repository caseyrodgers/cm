Admin Menu Changes

Manage Groups has been moved within Student Registration. Refresh List has been removed.

Time on Task

A time on task estimate is provided in Student Detail History, Highlights, Report Card and Export. The estimate is based on a standard time assigned to each resource accessed by a student; Quiz: 15 minutes, Flashcard: 10 minutes, Game: 10 minutes, Required Practice Activity: 10 minutes, Extra Practice Problem: 5 minutes, Quiz Results: 5 minutes, Required Practice Problem: 5 minutes, Video: 5 minutes, Whiteboard: 5 minutes, and Lesson: 3 minutes.

Parallel Programs

We have added a new feature called Parallel Programs for students to be assigned multiple separate programs within Catchup Math. For example, a student can be working on a Subject/Chapter Program in their regular math class, as well as a separate Proficiency Program in their math lab. Or, teachers can use this feature to assign extra work or a Custom Program or Quiz. Select the Parallel Programs button under Student Registration. Students access Parallel Programs by logging in with their school Login Name and the name of the Parallel Program as their initial password, and then providing their Personal Password. Student work is saved in Student Detailed History and is reflected in Report Cards, Export, and Highlights. Please contact your Account Manager for more details.

Feature Updates prior to July 30, 2011  <top>

Log in As a Student in "Teacher Mode"

Teachers can now log into any student account by right-clicking on a selected student from the summary listing. In teacher mode, there is a button to mark all quiz answers correct, and also there is no requirement to view all the required problems/activities in order to advance. This has many uses but was created primarily for convenience in reviewing new custom programs.

Change to Allowing Games

The Advanced registration option for restricting games now prevents the playing of games entirely. As the games are appreciated by students and they have a learning benefit, we suggest that you only restrict games for students who play them excessively.

Custom Quizzes

The Custom button now allows the creation of custom quizzes as well as custom programs. Teachers may select questions from our question bank to create a custom quiz or test. Note that you can tell students they are required to show their work on the whiteboard.

Common Core Alignment Table Available

A table is now available showing specific lesson correlation with Common Core standards. See our Math Educators page (easily found from the site map at the bottom of our Home page).

Past Quiz Results Now Available to Teachers

All quizzes taken by students are now available in Student Detail History.

Select Section Number of Program

Using the Advanced Options feature in Registration or Edit Student, it is now possible to select the section number of an assigned proficiency or graduation preparation program. This can be handy in special circumstances, e.g., a student accidentally clicked Check Quiz without submitting any answers and you want to let them start over.
